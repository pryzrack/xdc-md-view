### Enlazar Ficheros

Para enlazar un archivo markdown con otro, utilize `[[ texto | nombre ]]` (sin extensión). Todos los ficheros a enlazar deben estar dentro de la carpeta **data**.

#### Ejemplo:

```markdown
[[ foo | bar ]] Este es un archivo enlazado
```
