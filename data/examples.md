## Tutoriales de Programación

### Python

Bucle `while`:

```python
numero = 0
while numero < 10:
    numero += 1
    print(numero)
```

### JavaScript

Función recursiva:

```javascript
function factorial(n) {
    if (n === 0) {
        return 1;
    }
    return n * factorial(n - 1);
}
```

### Go

El clásico programa Hola mundo en Go:

```go
package main
 
import "fmt"
 
func main()
{
  fmt.Printf("Hola, mundo\n")
}
```
